# FrameKit

Modular C/C++ framework ecosystem with a contracts-first core and optional transport/media toolkits.

## What This Organization Contains
- `framekit` provides core runtime contracts and multiprocess application framework behavior.
- `netkit` and `mediakit` are optional ecosystem modules for transport and media domains.
- Repositories follow shared governance doctrine for consistency and maintainability.

## Repositories

### Active
- [framekit](https://github.com/tinmanworks-framekit/framekit) - Modular C/C++ application framework with multiprocess runtime contracts and optional backends.
- [netkit](https://github.com/tinmanworks-framekit/netkit) - Optional IPC and networking transports for FrameKit.
- [mediakit](https://github.com/tinmanworks-framekit/mediakit) - Optional media session and pipeline toolkit for the FrameKit ecosystem.

### Legacy
- [framekit-legacy](https://github.com/tinmanworks-framekit/framekit-legacy) - Legacy/maintenance codebase retained for reference; new development should target `framekit`.

## Governance
Governance and policy references are maintained in [tinmanworks-foundation/doctrine](https://github.com/tinmanworks-foundation/doctrine).

## Getting Started
1. Start with [framekit](https://github.com/tinmanworks-framekit/framekit).
2. Add [netkit](https://github.com/tinmanworks-framekit/netkit) for IPC/network transport capabilities.
3. Add [mediakit](https://github.com/tinmanworks-framekit/mediakit) for media session and pipeline extensions.

## Contribution
Use issues in the relevant repository for fixes, feature requests, and governance-impacting proposals.
